在開始之前先大致介紹一下台灣目前光纖網路接取技術使採用 GPON (Gigabit Passive Optical Network) 千兆級無源光網路，簡單架構如上圖


不知道大家有沒有曾經好奇過從數據機到機房會經過哪些設備，這些設備分別又提供那些功能呢 ?


設備主要分散於三個地方:

  1. 中華電信各地區機房，又稱為局端
  2. 公有區域 包含: 路邊人行道上的機箱、大樓電信室
  3. 住家內

局端設備

  1. BRAS (Broadband Remote Access Server) 寬頻遠程接入伺服器

    是連接 ISP 邊緣網路 (Edge Network) 與局端機房的設備，主要提供 PPPoE 驗證用戶 (PPPoE Server)、發配 IP 等功能
    在 201x 年左右網路上有 power user　在論壇討論 BRAS 路由與出國頻寬的關係
    Hinet用戶 如何查詢自己Bras路由+如何查詢自己在中華電信的電路號碼
    HiNet 寬頻上網用戶連線資訊查詢服務

    註: 光世代後目前鮮少看到用戶討論更換 BRAS 路由，很有可能是 CDN 技術普遍再加上兩大影視巨頭 Youtube Netflix 均有在台灣架設機房
    Youtube: 彰濱 GCP 機房 + Google Global Cache (GGC)
    Netflix 攜手 HiNet 在台建置 CDN
    不過用戶若有其他服務需要穩定出國，有更換 BRAS 路由這招 (假設目前的不夠穩定)

    補充: PPPoE Server 還有限速的功能，不然 OLT 整條頻寬都給你用怎麼可以 PPPoE Server 實作 - 宿網管理(自己當小小ISP)


  1. OLT (Optical Line Terminal) 光纖線路終端



    圖片來源 & Wiki 介紹

    從 OLT 一條光纖線路拉出去就是負責幾百幾千戶的寬頻匯接，在把封包資料送到 BARS 進入 Edge Network。 OLT 除了匯集光纖線路外還有驗證 ONU (光纖數據機) S/N 的功能，提供初步用戶驗證

公有區域 中華電信人行道上的機箱

參考資料 + 圖片來源


機箱內部圖

左邊黑色外皮的是從骨幹拉出來的來源線，接到光分器 (optical splitter)，綠色插座即連接各戶 ONU (光纖數據機)

住家內

住家內設備就 ONU 光纖數據機一台啦! 早期中華電信給的機台是 i040gw，近期給的機台換成 H660WM/ H660W
數據機主要提供的功能就是光電轉換，將來源光訊號轉換成電子訊號 (RJ-45)
